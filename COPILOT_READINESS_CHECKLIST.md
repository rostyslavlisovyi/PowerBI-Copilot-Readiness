# Microsoft Copilot Readiness Checklist

> Working document for assessing and improving Power BI semantic models using Microsoft guidance and project experience.

---

## Purpose

This checklist is used to assess whether a Power BI semantic model is ready for effective use with Microsoft Copilot.

The first version is based primarily on Microsoft documentation and best practices.

Additional project-specific recommendations may be added only after they have been validated during real implementations.

---

## Scope

This assessment covers:

- Semantic models
- Tables
- Columns
- Measures
- Relationships
- Metadata
- Documentation

The assessment does not cover report design, visuals, Power BI administration or infrastructure.

---
