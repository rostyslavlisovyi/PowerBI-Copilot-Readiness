# PowerBI-Copilot-Readiness

Project for assessing, documenting and transforming Power BI semantic models to achieve Microsoft Copilot Ready standards, ensuring complete compatibility with Copilot through high-quality metadata, semantic modeling, business context and AI-ready documentation.

## GitHub Copilot Instructions

This repository includes custom [GitHub Copilot instructions](.github/copilot-instructions.md) that guide Copilot to follow the Power BI semantic model standards, DAX guidelines, documentation requirements and safety rules defined for this project.
